# 💄 Glow Beauty Salon Website

## 📌 Project Overview

Glow Beauty Salon Website is a modern and responsive business website developed for a local beauty salon. The website helps customers explore services, learn about the salon, and contact the business for appointments.

This project was developed as part of the Full Stack Web Development Internship at Future Interns.

---

## 🚀 Features

- Responsive and modern user interface
- Home page with business introduction
- Services section
- About Us section
- Contact form
- Appointment booking button
- Mobile-friendly design
- Professional navy blue theme

---
